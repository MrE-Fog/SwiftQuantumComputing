  • oracle(truthTable:targets:controls:)

    Produces a list of Gate.oracle(truthTable:target:controls:). An oracle is produced for each target in targets if at least one activation in truthTable has the bit correspoding to the target activated.

    Declaration

    Swift

    public static func oracle(truthTable: [ExtendedTruth],
                          targets: [Int],
                          controls: [Int]) -> [Gate]

    Parameters

    truthTable

    List of Gate.ExtendedTruth instances.
    targets

    Targets for each oracle in the list.
    controls

    Control qubits for each oracle in the list.

    Return Value

    List of Gate.oracle(truthTable:target:controls:).
